# Material Management System — SNHRC

Enterprise hospital asset and material management system for **Sri Narayani Hospital & Research Centre**.

## Overview

A modern, scalable internal web application for managing medical assets, equipment, and materials across a multi-branch hospital network. Built to replace legacy form-based tracking systems with a structured, user-friendly platform.

## Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **Charts**: Recharts
- **Routing**: React Router v6

## Getting Started

```bash
npm install
npm run dev
```

The application will be available at `http://localhost:8080/`.

## Key Modules

- **Dashboard** — KPI overview with asset statistics and charts
- **Masters** — Branch and Department management
- **Asset Management** — Add assets and registry views
- **Tracking** — Barcode/serial number lookup
- **Transactions** — Material transfer workflow
- **Equipment** — Biomedical and general equipment views
- **Reports** — Filtered data export
